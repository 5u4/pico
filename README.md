<h1 align="center">pico</h1>

<p align="center">
  Your personal AI assistant, living in Discord.
</p>

---

pico is a self-hosted personal AI assistant. It does more than answer questions —
with a full set of tools it researches, writes, analyzes, and takes real action to
get things done for you. Every Discord thread is one ongoing conversation with its
own working directory and memory, so you can talk about anything and ask it to do
anything.
